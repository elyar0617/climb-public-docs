# Climb Public Docs

Public documentation pages for the Climb Android app.

This repository is intended for GitHub Pages hosting and should contain documentation pages only. It should not include Android source code, build files, signing files, or private project assets.

## Pages

- `index.md`
- `privacy-policy.md`
- `support.md`

## GitHub Pages Setup

1. Create a public GitHub repository named `climb-public-docs`.
2. Copy these markdown files into the repository root.
3. In GitHub, open Settings > Pages.
4. Set Source to the default branch and root folder.
5. After publishing, use this privacy policy URL format in Google Play Console:

```text
https://<github-username>.github.io/climb-public-docs/privacy-policy
```

## Before Publishing

- Replace `support@yourdomain.com` with the real support email when available.
- Confirm the effective date is correct.
- Re-review the policy whenever Climb adds login, cloud sync, analytics, ads, payments, bank connections, AI services, or third-party SDKs.
